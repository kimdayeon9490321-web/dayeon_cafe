# 카페 랜딩페이지 제작(원페이지)

## 요약
- 카페 방문/예약/메뉴 확인을 빠르게 유도하는 원페이지 랜딩을 제작
- 기본 섹션: 헤더 → 히어로 → 대표 메뉴 → 공간/서비스 → About → 위치/운영 → 문의 → 푸터
- 컨셉 3종(동네 단골형/디저트·시그니처형/작업·공간형) 프리셋 전환 지원
